## Introduction

### About

**Nectar UI** is an elegant and beautiful React component library with Geek style,
originating from [Vercel](https://vercel.com?utm_source=nectar-ui&utm_campaign=oss) design system.

With Nectar's unique advantages in text layout, optimization of light and dark lines, and scale components,
you can easily create clean and powerful personal sites, or quickly build modern web applications based on highly customizable design system.

Get start on Nectar with the [guide](/en-us/guide/installation),
or view full [project examples](https://github.com/nectar-org/nectar-ui/tree/master/examples).
